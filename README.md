# JAMES_source

This repository contains source code and Jupyter notebooks used for research on
tropical cyclone intensity prediction.

The repository is maintained by Jae-Deok Lee and is intended to support
scientific reproducibility, analysis documentation, and academic research.

This repository is associated with a manuscript prepared for the
**Journal of Advances in Modeling Earth Systems (JAMES)**.

## Repository contents

At present, this repository includes Jupyter notebooks for model development,
analysis, and evaluation.

Example contents may include:

- Preprocessing workflows
- Model training scripts or notebooks
- Model evaluation procedures
- Analysis of rapid intensification and rapid weakening
- Visualization and diagnostic routines

## Usage

The notebooks and scripts in this repository are provided for academic and
research purposes.

Before running the notebooks, please check the required Python environment,
input data paths, and package dependencies. Some notebooks may require external
datasets that are not included in this repository.
